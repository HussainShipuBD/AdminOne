    Button sizes and full width

    In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the .btn-outline-* ones to remove all background images and colors on any button.

    Simple code

                          
<button type="button" class="_btn _btn_outline_primary _btn_sm">Primary</button>
<button type="button" class="_btn _btn_outline_primary _btn_100 _btn_sm">Primary</button>

<button type="button" class="_btn _btn_outline_secondary _btn_100">Secondary</button>
